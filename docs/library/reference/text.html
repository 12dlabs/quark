Text utilities

There are lots of utilities for text.

AliHub.Common.Text

This is a powerful tools to provide lots of functions for text.

Functions

Format

You can format a string from a template and data by following function.

/**
  * Formats a string.
  * @param template The source string.
  * @param parameters The object or array of parameters.
  * @param urlEncode true if for URI components; otherwise, false.
  * @param preBag A property bag for preparing.
  */
public static format(
    template: string,
    parameters?: any,
    uriComponent = false,
    preBag: string[] | Func<string[]> = null
): string {}

Following is a sample to use.

var qText = AliHub.Common.Text;

// Format a normal string.
qText.format(
    "Hi {name}, what's your {property}?",
    { name: "Kingcean", property: "name" }
);

// Format a URL.
qText.format(
    "/api/sample/assets/list.html?q={q}&page={page}&size={size}",
    { q: "Hello Kitty", page: 0, size: 20 },
    true
);

// Format something with a definition
// to avoid some properties are not in the paramters.
qText.format(
    "{hours} {minutes} {seconds}",
    { hours: 4, minutes: 1 },
    false,
    ["hours", "minutes", "seconds"]
);

Trim

Left trim, right trim or both trim a given string by following method.

public static trim(str: string): string {}
public static leftTrim(str: string): string {}
public static rightTrim(str: string): string {}

Following is the sample.

var qText = AliHub.Common.Text;
var str = " Alibaba ";
var t1str = qText.trim(str); // "Alibaba"
var t2str = qText.leftTrim(str); // "Alibaba "
var t3str = qText.rightTrim(str); // " Alibaba"

Find

You can get the position of the specific strings in a given string.

/**
  * Returns the position of the first occurrence of a substring.
  * @param searchString The substring to search for in the string.
  * @param position The index at which to begin searching the String object.
    If omitted, search starts at the beginning of the string.
  * @param last true if search the last one of position items;
    otherwise, false.
  * @param negative true if search from last; otherwise, false.
  */
public static indexOf(
    str: string,
    position: string | number | (string | number)[],
    last = false,
    negative = false
): TextPositionContract {}

Following is the sample to use.

var qText = AliHub.Common.Text;
var str = "abcdefgXhijklmnXopqXrst";
// Find first of "bc", position 5, "X" in a given string.
// Result: { q: "bc", start: 1, end: 2 }
var info = qText.indexOf(str, ["bc", 5, "X"]);

// Find last of "bc", position 5, "X" in a given string.
// Result: { q: "X", start: 7, end: 7 }
var info = qText.indexOf(str, ["bc", 5, "X"], true);

// Get index of first of "bc", position 5, "X" in a given string,
// by searching from last to end.
// Result: { q: "b", start: 1, end: 2 }
var info = qText.indexOf(str, ["bc", 5, "X"], false, true);

// Get index of last of "bc", position 5, "X" in a given string,
// by searching from last to end.
// Result: { q: "X", start: 17, end: 17 }
var info = qText.indexOf(str, ["bc", 5, "X"], true, true);

Base64

You can encode or decode base64.

// Encode a string as Base64.
public static base64encode(str: string): string {}

// Decode a string by Base64.
public static base64decode(str: string): string {}

Following is the sample for usage.

var qText = AliHub.Common.Text;
var str = "Nice to meet you!";
// Get Base64 from a given string.
var base64str = qText.base64encode(str);

// Decode by Base64 to compare original one.
var isSame = qText.base64decode(str) === str;
