Quark

一个 JavaScript 库，可以帮助你创建稳定强大的富网站应用，并通过减少你的代码量来降低开发与维护成本。

  • MVC - 全新 DataSet 驱动的 Model，VisualControl 驱动的 View，ViewController 和 PageController 驱动的 Controller，三者合一，天衣无缝。
  • 增强型 MVVM 模式，包括数据模型、视图模板、视图扩展、视图模型。
  • 在移动端、桌面端、电视端都能自适应。
  • 内建异步模型加载器支持。
  • Web 组件支持。

那么，如何将 Quark 引入你的项目中去呢？只需在你的页面中直接引入以下代码。

<script type="text/javascript" src="https://g.alicdn.com/hub/quark/1.8.1000/scripts/index.min.js" charset="utf-8"></script>

然后你便可以直接使用 AliHub 命名空间下的功能了。

而对于 AMD 或 KISSY (Modulex) 用户，可以引入以下代码。

<script type="text/javascript" src="https://g.alicdn.com/hi/modulex/5.0.0/seed.js" charset="utf-8"></script>
<script type="text/javascript" src="https://g.alicdn.com/hi/quark/1.8.1000/index.js" charset="utf-8"></script>

然后，你就可以以下方式使用。其将返回一个功能集，也可以使用 AliHub 命名空间代替使用。

require("quark/script/import.min.js");

如果你在使用 Type Script，请确保使用 2.0 版本或以上，并可下载声明文件。

另外，Quark 还能与 jQuery、RequireJs、AngularJS、Knockout、Vue.js 等库紧密协同工作，并能扩展 KISSY (Modulex)、Flipper 等库的功能。
