Collection utilities

The utilities for collection and set provide lots of functions to access, convert and test a specific list. They are under AliHub.Collection namespace. See comment documentation for details.

Basic array operations

You can convert to, find in or modify an array by any of following ways.

// Gets an array by given item or array.
// This is used to work as a part of the processing logic
// that need support both single item or an array.
export function toArray<T>(
    value: T | T[],
    emptyForNull = true
): T[] {}

// Checks if the specific object is an array.
export function isArray(col: any): col is any[] {}

// Checks whether a specific list contains a test item.
export function contains<T1, T2>(list: T1[],
    testItem: T2,
    compare?: Common.Func2<T1, T2, boolean>
): boolean {}

// Gets the index of a test item in a specific list.
export function indexOf<T1, T2>(
    list: T1[],
    testItem: T2,
    compare?: Common.Func2<T1, T2, boolean>
): number {}

// Gets the element of a test item in a specific list.
export function getItem<T1, T2>(
    list: T1[],
    testItem: T2,
    compare?: Common.Func2<T1, T2, boolean>
): T1 {}

// Copies a list.
export function copy<T>(list: T[]): T[] {}

// Adds a list to target list one.
export function pushRange(
    list: any[],
    adding: any[],
    nullCheck = false
): any[] {}

// Removes a number of items from a target list.
export function remove(
    list: any[],
    removing: any[],
    changeRaw = false,
    compare: Common.Func2<any, any, boolean> = null
): any[] {}

// Formats a list.
export function formatList<T>(
    col: T[],
    format: AliHub.Common.Func1<T, T>
): T[] {}

String

You can change the items in a string array. Or do other things about string array.

// Modifies items in a specific list string.
export function addItem(
    raw: string,
    split: string,
    addingItems: string | string[],
    removingItems?: string | string[]
): string {}

// Converts to string array.
// This is used to work as a part of the processing logic
// that need support both single item or an array.
export function toStringArray(
    value: string | string[],
    emptyForNull = true
): string[] {}

    list: KeyValuePairContract<TKey, TValue>[],
    removing: TKey[],
    changeRaw = false
): KeyValuePairContract<TKey, TValue>[] {}

// Sets a key value pair.
export function setKeyValue<TKey, TValue>(
    col: KeyValuePairContract<TKey, TValue>[],
    item: KeyValuePairContract<TKey, TValue>,
    compare: Common.Func2<TKey, TKey,
    boolean> = null
): KeyValuePairContract<TKey, TValue>[] {}
