Samples

// Creates a list control.
var control = AliHub.Common.listControl(
    "demo_container",    // Container element ID.
    null,                // Parent control if has.
    {                    // Options.
        "areSame": "id",
        "isBefore": "created",
        "itemTemplate": itemTempl
    }
);

// Loads data via AJAX or JSONP. See related documentation of this website or its comments.
// pushResource is a method to push a set of collections,
// but only the list of which property is named by the specific one will be used for push range.
// pushResourceFromWeb is the method to load data by GET or JSONP to call pushResource.
// The 3rd parameter is the property name for property to push range the list.
control.pushResourceFromWeb("sample", "asset", "list");

// Clears the list.
control.clear();

The variable object control is the instance of class ListControl<T>.
