Development Environment Setup

To develop by Quark, you need setup the environment.

  • For Visual Studio users.
  • For other Type Script users.
  • For other ECMAScript 5 users.

For Visual Studio users

Using Visual Studio, developers can focus on development with an integrated environment with programming, building, deployment, etc.

Requirement

  • Microsoft Visual Stuido 2015 Update 3.
  • Type Script 1.8.4 or higher.

For the users who are not using Windows 10, following are also required.

  • Powershell 4.0
  • Git client

Downloading and installing

You can download from any of following URLs.

  • Alibaba CDN
    http://g.alicdn.com/hub/quark/1.8.1000/scripts/index.min.js

You can also import this directly. For details about import, please read KB article How to import to my project.

For other Type Script users
