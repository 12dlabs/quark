There are lots of utilities for observable, task and reflection.

Binding and observable

You can create a binding and observable object or array to watch the changes.

namespace AliHub.Common {
    function bindingObj<T>(value?: T): BindingObjectContract<T> {}
    function bindingArray<T>(value?: T): Collection.BindingArrayContract<T> {}
}

See comment documentation of AliHub.Common.bindingObj and AliHub.Common.bindingArray for details.

Tasks

Provide a way to manage task. You can get a handler to manage a continuity task.

namespace AliHub.Common {
    function getContinuityTask(options: ContinuityTaskOptionsContract): Action {}
}

See comment documentation of AliHub.Common.getContinuityTask and AliHub.Common.ContinuityTaskOptionsContract for details.

Samples

var qCore = AliHub.Common;

// Creates a function that can handler the task configured by you for continuity task.
// See AliHub.Common.getContinuityTask(options) function for details.
var hitProc = qCore.getContinuityTask({
    start: 3, // The count clicked to occur specific handler at least.
    count: 100, // The count to allow occur at most.
    span: 400, // The time span in millisecond between two clicks at most.
    process: function (count, options) { // A handler to occur.
        // count is the count clicked currently now.
        // options is the object of this.
    }
});

// It will return a function so that you can call it anywhere you want to process, such as clicking.
$("#idOrElementOfTheDomToBind").on("click", function ()
    hitProc();
});
