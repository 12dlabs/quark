AliHub.Graph.ImageContract

namespace AliHub.Graph {
    export interface ImageContract { }
}

This is a contract for rendering image in different ways. It supports numbers of image type.

  • Picture.
    See comment documentation of AliHub.Graph.UrlImageContract.
  • SVG.
    See comment documentation of AliHub.Graph.SvgImageContract.
  • String.
    See comment documentation of AliHub.Graph.StringImageContract.

Samples

To resolve the image element (perhaps it is not an IMG element), you can follow the following sample.

// Creates an image info.
var info = {
    "type": "url",
    "url": "http://quark.alibaba.net/inc/images/figures/architecture_mvc.png"
};

// Gets the element.
var image = AliHub.Graph.ImageContainer.generateImage(info);

In view template, it can be rendered by image() function of root. Suppose property icon is an image in view model.

<div data-bind="image: model().icon"></div>
