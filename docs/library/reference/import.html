How to import to my project

You can add following code to insert the library directly.

<script type="text/javascript" src="https://g.alicdn.com/hub/quark/1.7.891/scripts/index.min.js" charset="utf-8"></script>

But you can also import Quark by asynchronous modules loader as well as static script reference. And this is recommended. Quark supports following ways to load async, please select the one used in your project.

  • AMD (e.g. RequireJs)
  • Modulex / KISSY 5 / KISSY 6

AMD

Quark works well by Asynchronous Module Definition (AMD) with RequireJs. Please just put the file in the directories for references, or import by loader config.

RequireJs can be downloaded from http://requirejs.org/docs/download.html.

Modulex / KISSY 5 / KISSY 6

Firstly, adds following script in the page.

<script type="text/javascript" src="https://g.alicdn.com/hi/quark/1.7.891/index.js" charset="utf-8"></script>

And reference it and setup by following script.

require("quark-bindings/script").setup();

KISSY is developed by Alibaba Group. Modulex is a folk version from KISSY 5. For details about KISSY and Modulex, please visit the related documentation.
