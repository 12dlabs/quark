You can render and manage a control to record the path user walks through. This is useful for single page and other senarios like the one when we need turn back or control what should be displayed.

Samples

// Creates a activity control.
var control = AliHub.Common.activityControl(
    "demo_container",    // Container element ID.
);

// Adds a control as one of steps.
var firstPanel = control.add();
firstPanel.appendElement("div").innerHTML = "First page";

// Adds another control as one of steps.
var secondPanel = control.add(function (cid) {
    var childControl = new AliHub.Common.VisualControl(cid);
    childControl.appendElement("div").innerHTML = "Second page";
});

// Turns back.
control.back();

The variable object control is the instance of class ActivityControl.
